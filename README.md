Доп.задание
Описание:Позволяет добавлять бизнес-процессы с указанием их названия, типа и рейтингов по операционным и стратегическим критериям.Автоматически рассчитывает для каждого процесса значения эффективности и важности.Определяет зону, в которую попадает процесс, основываясь на расчетных значениях .Визуально отображает процессы в таблице и распределяет их по соответствующим зонам матрицы.Позволяет фильтровать процессы по зонам.
